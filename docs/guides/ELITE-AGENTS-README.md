*arq  # Architecture and system design
*orc  # Project orchestration and coordination  
*zen  # Clean code and elegant programming
*vex  # Creative design and user experience
*sage # Strategic analysis and market intelligence
*nova # Innovation and breakthrough thinking  
*echo # Community building and authentic voice

### Installation Methods

#### Method 1: Global Installation
```bash
cd ~/Desktop/agents
./install.sh
# Restart terminal, then use anywhere:
*arq "Design a microservices architecture for my fintech platform"
```

#### Method 2: Project-Specific Installation
```bash
cd your-project
# Copy specific agents you need
cp ~/Desktop/agents/ARQ.md ./
cp ~/Desktop/agents/ZEN.md ./
cp ~/Desktop/agents/VEX.md ./
```

#### Method 3: Claude Code Integration
Add to your project's `CLAUDE.md`:
```markdown
# Available Elite Agents
- ARQ (Architecture): ~/Desktop/agents/ARQ.md
- ZEN (Clean Code): ~/Desktop/agents/ZEN.md  
- VEX (Design): ~/Desktop/agents/VEX.md
- SAGE (Strategy): ~/Desktop/agents/SAGE.md
```
